## âš™ï¸ Working Principle
The ultrasonic sensor continuously measures the distance to obstacles.
If an obstacle is detected within a predefined threshold:
- The robot stops
- Moves backward dynamically
- Scans left and right using a servo-mounted sensor
- Chooses the safest direction and continues movement

Bluetooth control allows manual navigation when required.
